Frequently
Asked Questions

How can I contact support if I have
questions?

+

Please contact us at @zubalenok.team on Instagram or support_zubalenok@zbl.agency, and we'll assist you right

I can barely touch my toes. Is this too advanced for me?

+

Not at all. 80% of students start exactly where you are. The program meets you at your level.

Can I use the course on my tablet / PC / TV?

+

The methods for streaming videos to your TV depend on the type of device and the platform you’re registered on. Most commonly, you can use one of the following options:

1. Through a Smart TV browser
2. Through your phone or laptop + Smart TV (using the “Cast” or “Screen Mirroring” function)
3. Through an HDMI cable

Access from a PC is also available, simply open the link to the platform from your email on your computer.

Can I get a refund if I don't like the course?

+

Yes, we offer a 14-day money-back guarantee if you’re not satisfied.

What are the terms of the money-back guarantee?

+

Just mail support_zubalenok@zbl.agency or DM @zubalenok.team. Provide your Name, Surname, invoice and we will help you out. Refunds take 5-10 days to get proceeded.

Do I need special equipment to participate in the course?

+

You'll only need comfortable workout clothes, a mat for exercises and yoga blocks to make workouts easier.

I don't have yoga blocks. What can I use instead?

+

Thick books, sturdy boxes, stacks of folded towels, or firm cushions all work.

Is this the same as Only Flexibility 2.0?

+

No It’s a brand new course. Upgraded version with refined techniques, better structure, and new bonus content based on our students feedback and wishes.

Is the course suitable for pregnant women?

+

Pregnant women are strongly advised to consult with a doctor before starting the course. While many exercises are safe, it’s important to ensure they are suitable for you.

What are the contraindications for participation?

+

If you have chronic diseases or have recently undergone surgery, please consult a doctor before starting the course. Exercises should not cause pain or discomfort.

How can I get the most out of the course?

+

Follow Karyna’s recommendations, perform the exercises regularly (4-6 times a week), and remember the importance of proper nutrition and rest. Regular practice will lead to the best results.

I'm 40+. Will this work for me?

+

Age doesn't matter. We have students in their 50s and 60s making incredible progress.

@use "sass:math";

@mixin resize(
  $min,
  $max,
  $property: font-size,
  $min-vw: 340px,
  $max-vw: 1900px
) {
  // Видалення одиниць вимірювання
  $min-no-unit: math.div($min, ($min * 0 + 1));
  $max-no-unit: math.div($max, ($max * 0 + 1));
  $min-vw-no-unit: math.div($min-vw, ($min-vw * 0 + 1));
  $max-vw-no-unit: math.div($max-vw, ($max-vw * 0 + 1));

  // Перетворення в REM
  $min-rem: math.div($min-no-unit, 16);
  $max-rem: math.div($max-no-unit, 16);

  // Розрахунок формули для плавної зміни
  $slope: math.div(
    $max-rem - $min-rem,
    math.div($max-vw-no-unit - $min-vw-no-unit, 16)
  );
  $y-axis: -1 * $min-vw-no-unit * math.div($slope, 16) + $min-rem;

  // Динамічне ім'я властивості
  #{$property}: clamp(
    #{$min-rem}rem,
    #{$y-axis}rem + #{$slope * 100}vw,
    #{$max-rem}rem
  );
}

const questions__main = () => {
  const init = () => {
    const items = document.querySelectorAll(".questionsContainer-item");
    if (!items.length) return;

    items.forEach((item) => {
      const top = item.querySelector(".questionsContainer-item-top");
      const bottom = item.querySelector(".questionsContainer-item-bottom");
      const label = item.querySelector(".label");

      if (!top || !bottom || !label) return;

      bottom.style.maxHeight = "0";
      bottom.style.overflow = "hidden";
      bottom.style.transition = "max-height 0.4s ease";

      top.addEventListener("click", () => {
        const isActive = item.classList.contains("active");

        items.forEach((el) => {
          el.classList.remove("active");
          const elBottom = el.querySelector(".questionsContainer-item-bottom");
          const elLabel = el.querySelector(".label");
          if (elBottom) elBottom.style.maxHeight = "0";
          if (elLabel) elLabel.textContent = "+";
        });

        if (!isActive) {
          item.classList.add("active");
          bottom.style.maxHeight = bottom.scrollHeight + "px";
          label.textContent = "–";
        }
      });
    });
  };

  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', init);
  } else {
    init();
  }
};

export default questions__main;
